🚪 API Gateway Service
Входные ворота системы Event Bridge — единственный сервис, с которым напрямую взаимодействует внешний мир.
📋 Описание
API Gateway отвечает за приём заявок на регистрацию мероприятий, их валидацию и публикацию событий в RabbitMQ для дальнейшей обработки другими сервисами системы.
🔹 Зона ответственности
✅ Приём HTTP-запросов на регистрацию
✅ Валидация входных данных
✅ Обогащение событий метаданными (registration_id, registration_time)
✅ Публикация событий в RabbitMQ с правильными routing keys
❌ Не обрабатывает события (это делают другие сервисы)
❌ Не знает о потребителях сообщений
